# Wave 8 — Empirical Studies

Snodo paper Section 5 studies: Monte Carlo on disagreement policies,
detection probability, byzantine robustness, and overhead benchmarks.

## Quick start

```
pip install -r studies/requirements.txt
make studies          # run all studies headless
make study NAME=_smoke  # run just the smoke test
```

Interactive editing (browser):
```
marimo edit studies/_smoke/notebook.py
```

Headless (no browser):
```
python studies/_smoke/notebook.py
```

## Architecture

```
studies/
  _common/           # Shared helpers (seeded RNG, I/O, plotting)
  <study_name>/
    notebook.py      # marimo notebook (also runs as a script)
    outputs/         # SVG figures + CSV data (regenerated by make)
```

See `conventions.md` for the seed policy, output naming, and paper handoff rules.
